# React Redux Hook Template

## Current Features
This is a boiler plate for starters, more features will be added.
* React
* Redux
* Redux Hooks
* Redux Thunk Middleware
* SCSS support
* Jest
* React Testing Library
* Webpack 4
* Webpack Dev Server
* Redux Devtools
* ESLint configuration
* JS, CSS Code minification
* css

## Running the project
```
yarn install
```

```
yarn start
```
